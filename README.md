This is Webpack Template.(Webpack/Typescript/React/Sass/Browser-sync/gulp)

## Installation

    $ git clone https://github.com/115design/webpackTemplate.git
    $ npm install
    $ export PATH=$PATH:./node_modules/.bin
    $ typings install
    $ npm run start
